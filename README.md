# AstrBot 古诗插件 (astrbot_plugin_ancient_poem)

这是一个 AstrBot 插件，用于随机获取并发送一句古诗词。

## 功能

- **指令**：`/gs`
- **功能**：随机获取一句古诗词，并附带作者和出处。
- **数据来源**：[一言 (Hitokoto)](https://hitokoto.cn/) API (诗词分类)。

## 安装

1. 将 `astrbot_plugin_ancient_poem` 文件夹放入 AstrBot 的 `data/plugins/` 目录。
2. 重启 AstrBot 或重新加载插件。

## 依赖

- aiohttp

## 示例

**用户**：/gs
**机器人**：
「众里寻他千百度。蓦然回首，那人却在，灯火阑珊处。」
—— 辛弃疾《青玉案·元夕》
